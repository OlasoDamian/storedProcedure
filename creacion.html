Creación de Procedimientos en MySQL

Para crear un procedimiento almacenado en MySQL, utilizamos la sentencia CREATE PROCEDURE. Aquí hay algunos ejemplos de cómo se crean procedimientos almacenados:

Procedimiento Básico


DELIMITER //
CREATE PROCEDURE ejemploBasico()
BEGIN
    SELECT 'Hola, Mundo!';
END //
DELIMITER ;

Procedimiento con Parámetros

Los procedimientos pueden aceptar parámetros para ser más dinámicos. Aquí se muestra cómo:


DELIMITER //
CREATE PROCEDURE saludar(IN nombre VARCHAR(50))
BEGIN
    SELECT CONCAT('Hola, ', nombre, '!');
END //
DELIMITER ;
