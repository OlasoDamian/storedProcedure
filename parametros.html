Uso de Parámetros en Procedimientos Almacenados

Los procedimientos almacenados en MySQL pueden aceptar diferentes tipos de parámetros para mayor flexibilidad:

Parámetro IN

Usado para pasar valores al procedimiento:


DELIMITER //
CREATE PROCEDURE obtenerEmpleado(IN empID INT)
BEGIN
    SELECT nombre FROM Empleados WHERE id = empID;
END //
DELIMITER ;

Parámetro OUT

Usado para devolver valores desde el procedimiento:


DELIMITER //
CREATE PROCEDURE obtenerSueldoTotal(OUT sueldoTotal FLOAT)
BEGIN
    SELECT SUM(sueldo) INTO sueldoTotal FROM Empleados;
END //
DELIMITER ;

Parámetro INOUT

Usado tanto para recibir como para devolver valores:


DELIMITER //
CREATE PROCEDURE actualizarSueldo(INOUT empID INT, INOUT nuevoSueldo FLOAT)
BEGIN
    UPDATE Empleados SET sueldo = nuevoSueldo WHERE id = empID;
    SELECT sueldo INTO nuevoSueldo FROM Empleados WHERE id = empID;
END //
DELIMITER ;
